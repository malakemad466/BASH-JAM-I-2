# G-Bash
### By: Mostafa Medhat Ahmed

## Description
A terminal-based multi-game application written in Bash scripting.
The program allows users to choose between different interactive games from a main menu.
It demonstrates the use of arrays, functions, loops, conditionals, and input validation in Bash.

---

## Games Included

### Guess The Word Game
Guess a hidden word selected from a predefined list.

**Input:**
- Choose a number to select a word.
- Enter word guesses (5 attempts).
- Optional: Type `h` to use a hint (description or letter hint, once each).

**Output:**
- Displays number of correct letters in correct positions.
- Win message if guessed correctly.
- Reveals the correct word if attempts reach zero.

---

### Password Guess Game
Two-player turn-based password guessing game.

**Input:**
- Two players enter their names.
- Each player selects a secret 4-digit password (1000–9999).
- Players take turns guessing the opponent’s password.

**Output:**
- Displays number of correct digits in correct positions.
- Announces the winner when all 4 digits match.

---

### Reverse Words Game
Guess the correct word from a reversed version.

**Modes:**
- Linux Commands
- Linux Distributions

**Input:**
- Choose mode (Commands or Distros).
- Select a reversed word.
- Enter guesses (5 attempts).

**Output:**
- Displays remaining guesses.
- Shows correct answer with description upon win or loss.
